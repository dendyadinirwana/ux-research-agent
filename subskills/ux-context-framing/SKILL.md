# UX Context Framing

## Output wajib
- Research objective (1 kalimat)
- Target user utama + secondary
- Business objective
- Facts vs assumptions list
- 3 open questions

## Prompt inti
1. Why now?
2. Siapa yang paling terdampak?
3. Apa risiko kalau tidak diselesaikan?
4. Constraint utama (waktu, tim, budget)?

## Exit criteria
- Objective jelas
- User scope jelas
- Assumption terlabel
