# UX Prototype Planning

## Output wajib
- Learning goals (LG1, LG2, ...)
- Prototype fidelity choice + reason
- Test tasks draft
- Risk assumptions to validate

## Fidelity guide
- Sketch: concept/flow
- Lo-fi: IA/navigation
- Hi-fi: UI/interaction
- Functional: technical feasibility

## Exit criteria
- Prototype objective clear
- Test-ready plan available
