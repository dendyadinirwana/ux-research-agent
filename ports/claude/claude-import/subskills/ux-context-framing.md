# UX Context Framing

## Required outputs
- Research objective (1 kalimat)
- Target user utama + secondary
- Business objective
- Facts vs assumptions list
- 3 open questions

## Core prompts
1. Why now?
2. Who is most impacted?
3. What is the risk if this is not solved?
4. What are the key constraints (time, team, budget)?

## Exit criteria
- Objective is clear
- User scope is clear
- Assumptions are labeled
