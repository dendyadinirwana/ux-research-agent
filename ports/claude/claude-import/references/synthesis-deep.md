# Deep Synthesis Guide (English Edition)

Use this reference to convert raw qualitative data into decision-ready insights.

## Synthesis Levels
1. **Observation** — what happened
2. **Pattern** — what repeats
3. **Insight** — why it matters
4. **Implication** — what to change

## Workflow
1. Collect evidence snippets (quotes, events, actions)
2. Open-code each snippet
3. Cluster into themes (affinity)
4. Name each theme clearly
5. Draft insight statements
6. Prioritize opportunities

## Strong Insight Template
"[User segment] does [behavior], not because [common assumption], but because [underlying reason]. Therefore, [design/product implication]."

## Prioritization Heuristic
Score each opportunity:
- Severity (1–5)
- Frequency (1–5)
- Strategic fit (1–5)
Total = priority score

## HMW Conversion
From each validated insight, generate 2–3 HMW prompts:
- Solve directly
- Reduce friction indirectly
- Reframe the experience

## Output Contract
- Theme map
- Top insights (3–5)
- Opportunity ranking
- Confidence per insight
- Open questions for next cycle
