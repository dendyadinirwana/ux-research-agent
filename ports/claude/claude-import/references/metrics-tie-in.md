# Metrics Tie-In (English Edition)

Use this reference to connect qualitative insights to measurable outcomes.

## Why Tie Qual + Quant
- Qual explains behavior and causes
- Quant measures scale and impact
- Together they improve decision quality

## HEART Lens
- Happiness
- Engagement
- Adoption
- Retention
- Task Success

Map each research question to one HEART metric.

## Funnel Interpretation
For each funnel drop:
1. Quantify where loss happens
2. Investigate behavioral cause
3. Validate with targeted research
4. Define measurable fix

## Hypothesis Bridge
Template:
"If we change [X], then [metric Y] should improve by [Z], because [research evidence]."

## Metric Selection Rules
- Use one primary metric per decision
- Add 1–2 guardrail metrics
- Define baseline and target before implementation

## Output Contract
- Insight-to-metric map
- Baseline values
- Target values
- Measurement window
- Decision threshold
